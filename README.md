# parvalabs/tools

Homebrew tap for tools published by [parvalabs](https://github.com/parvalabs).

## Usage

```bash
brew tap parvalabs/tools
brew install local-slack
```

Or in one line, without a separate tap step:

```bash
brew install parvalabs/tools/local-slack
```

## Tools

| Formula | Description |
| --- | --- |
| [`local-slack`](Formula/local-slack.rb) | A local, throwaway Slack for developing and testing Slack apps/bots — [parvalabs/local-slack](https://github.com/parvalabs/local-slack) |
